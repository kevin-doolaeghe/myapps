# Node-RED - Automation

:triangular_flag_on_post: Setup the **Node-RED** automation stack.

## Author

**Kevin Doolaeghe**

## Setup

```
sudo docker stack deploy -c docker-compose.yml node_red
```

:warning: This program require a Docker instance **with Swarm mode** enabled to be executed.
